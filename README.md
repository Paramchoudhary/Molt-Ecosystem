# Molt-Ecosystem

A brutalism-style comparison website for the molt ecosystem, allowing users to browse, filter, and compare all projects built using the Moltbook API.

## Features

- **18+ Projects** - Comprehensive directory of molt ecosystem projects
- **Grid View** - Beautiful card-based layout with unique colors per project
- **Compare Mode** - Side-by-side feature comparison of selected projects
- **Advanced Filters** - Filter by category, status, engagement level, and source type
- **Search** - Real-time search across project names, descriptions, and features
- **Mobile Responsive** - Fully responsive design with mobile-first approach
- **Favicon Fetching** - Automatic favicon loading from project URLs

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) to view the site.
